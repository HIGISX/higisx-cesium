SuperMap iClient3D for WebGL/WebGPU — 三维GIS客户端开发平台

基于 WebGL/WebGPU 技术实现的三维GIS客户端开发平台，
可用于构建无插件、跨操作系统、 跨浏览器的三维 GIS 应用程序，可访问SuperMap iServer发布的多种服务、 实现了三维场景可视化浏览、交互操作和地理信息管理等功能。
通过本产品可快速实现浏览器和移动端上（WebGPU暂不支持）美观、流畅的地图呈现与空间分析。
SuperMap iClient3D 2023从底层同时支持WebGL和WebGPU两种渲染引擎，对外使用一套API接口便于用户根据实际需求快速切换，降低迁移成本。

SuperMap iClient3D for WebGL
GIS网络客户端开发平台

基于现代 Web 技术栈全新构建，是 SuperMap 云 GIS 和在线 GIS 平台系列产品的统一 JavaScript 客户端。集成了领先的开源地图库和可视化库，且核心代码以 Apache License 2.0 协议完全开源，连接了 SuperMap 与开源社区。提供了全新的大数据可视化和数据流可视化功能，通过本产品可快速实现浏览器和移动端上美观、流畅的地图呈现与空间分析。

  • 三维场景呈现
  • 高真实感三维特效
  • 三维地理空间分析

三维场景呈现

基于物理的渲染（PBR）材质、屏幕空间反射、景深等多种特效，提升整体可视化效果，使得三维场景更加真实美观。

高真实感三维特效

提供了多种高真实感的视觉特效，如：强大的粒子系统，能模拟雨天、雪天效果，泛光、扫描线等后处理特效、自发光、特效线等， 使场景呈现更生动、绚丽的效果。

三维地理空间分析

提供丰富的三维空间分析功能，实现复杂的三维空间分析。

高真实感三维特效

提供功能完善的接口，并为接口配备详细的API帮助文档， 使用户对接口的使用变得更加简单易用，同时提供更多功能支持
